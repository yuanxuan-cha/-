# 半页故事 - 网络小说阅读平台

## 项目概述

本项目是《前端设计技术课程设计》的主题二作品，以"半页故事"为原型，使用 Vue 3 + Vant 4 开发的网络小说阅读网站。项目参考了起点中文网的设计风格，提供了完整的小说浏览、分类、排行榜、书架等功能。

## 技术栈

- **框架**: Vue 3 (Composition API)
- **UI 组件库**: Vant 4
- **路由**: Vue Router 4
- **构建工具**: Vite
- **Mock 数据**: JSON Server
- **样式**: CSS3 + Vant 样式变量
- **HTTP 客户端**: Axios

## 核心功能详解

### 图片懒加载与性能优化

**实现位置**: `src/utils/index.js`

主要功能：
- 使用 IntersectionObserver API 实现图片懒加载
- 图片并发请求控制（最多4个同时请求）
- 图片缓存机制，避免重复加载
- 懒加载占位图与渐入动画

使用方式：
```vue
<img v-lazy="imageUrl" alt="图片描述" />
```

### 响应式设计

项目支持多断点适配：
- **PC端**: > 1024px
- **平板端**: 768px - 1024px
- **移动端**: < 768px
- **小屏移动端**: < 480px

### 本地存储功能

**收藏功能**:
- 收藏数据存储在 `localStorage` 的 `favorites` 键中
- 支持添加/取消收藏
- 书架页面展示收藏的小说

## API 接口设计

### Mock 数据接口 (JSON Server)

- `GET /banners` - 获取轮播图数据
- `GET /novels` - 获取小说列表（支持分类、类型筛选）
- `GET /novels/:id` - 获取小说详情
- `GET /ranks` - 获取排行榜数据
- `GET /recommends` - 获取推荐列表
- `GET /categories` - 获取分类列表
- `POST /login` - 用户登录
- `POST /register` - 用户注册
